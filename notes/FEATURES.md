# Fundamental Features
1. family joining and account management system
2. calendar/scheduling system
3. budget tracking
4. meal planning and grocery list generation
5. chore management system

## Family Joining and Account Management System
- One parent creates an account and a family
- A uniquie code is created for the family
- The other parent and children can join the family using the code
- Each family member can have their own account
- Each family member can have their own profile
- Children can see their chores, the family calendar, and the meal plan
- Parents can see all family member's chores, the family calendar, the meal plan, and the monthly budget for the family (parents may choose whether or not their children can see the budget)

## Calendar System
- Parents and children can add events to the calendar, and these events are visible to all family members by default but can be set to not visible to certain family members
- Events can be given a title, description, start date, end date, priority level (customizable by parents), type of event which sets its color (customizable by parents), recurring event (customizable by parents), notifications (customizable by parents)
- The family calendar can be viewed in a daily, weekly, or monthly view

## Budget Tracking
- Parents can create budget categories and set a monthly budget for each category
- Auto tracks payments similar to other budget tracking apps

## Meal Planning
- Drag and drop meals into a weekly calendar view
- Generate a grocery list from the meal plan
- Add recipes to the recipe book (this is where recipes are stored)
- Link meal plan to calendar system, so that meals appear on the family calendar
- Link meal plans to recipes (this is optional, but recommended)

## Chore Management
- Parents can create chores and assign them to family members
- Chores can be given a title, description, due date, priority level (customizable by parents), type of chore which sets its color (customizable by parents), recurring chore (customizable by parents), notifications (customizable by parents)
